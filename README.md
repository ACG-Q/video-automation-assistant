# 视频自动化助手

cdeaa.com 学习平台自动化 Chrome 扩展，支持自动答题、视频自动播放、任务队列管理。

## 功能特性

- 自动答题 -- 基于本地题库匹配题目答案，支持精确匹配、Jaccard 相似度匹配、关键词匹配三层策略
- 远程题库同步 -- 支持从远程 URL 同步题库，自动去重合并
- 手动选择答案 -- 题库未匹配时，弹出窗口手动选择，答案自动保存到题库
- AI 辅助 -- 复制题目和选项到剪贴板，方便粘贴到 AI 工具分析
- 视频自动播放 -- 自动识别页面视频，支持倍速控制、自动恢复播放
- 倍速守护 -- 视频加载后自动恢复设定的倍速
- 任务队列 -- 从课程列表批量抓取视频任务，按顺序自动播放
- 答题纠错 -- 监听正确答案显示，自动对比并修正已选答案

## 安装

1. 从 Releases 页面下载最新版本的 zip 文件
2. 解压到本地目录
3. 打开 Chrome 浏览器，进入扩展管理页面 (chrome://extensions)
4. 开启右上角的"开发者模式"
5. 点击"加载已解压的扩展程序"，选择解压后的目录

## 使用

### 个人中心页

打开 cdeaa.com 个人中心页面，点击扩展图标打开侧边栏：

- 点击"抓取"获取课程列表中的所有视频任务
- 点击"追加"在不重置队列的情况下追加新任务
- 查看用户信息和课程有效期

### 课程播放页

进入课程播放页面后自动启动：

- 自动检测并绑定视频元素
- 自动恢复倍速设置
- 自动匹配并回答题目
- 自动点击"继续观看"按钮进入下一节

### 侧边栏控制

- 任务选项卡：查看队列进度、暂停/恢复队列、跳转到指定视频
- 播放选项卡：控制视频播放/暂停、快进/快退、调节倍速
- 查看答题匹配结果和日志

### 手动选择答案

当题库未匹配到答案时，侧边栏会显示"手动选择"按钮，点击后弹出选择窗口，手动选择正确答案后自动保存到题库。

## 配置

在侧边栏点击设置图标打开配置页：

| 配置项 | 说明 |
|--------|------|
| 题干选择器 | 题目容器的 CSS 选择器 |
| 选项选择器 | 选项容器的 CSS 选择器 |
| 答题轮询间隔 | 检测新题目的间隔时间（毫秒） |
| 重试次数 | 未匹配时重新查询的次数 |
| 默认倍速 | 视频播放的默认倍速 |
| 快进/快退步长 | 每次快进或快退的秒数 |
| 自动答题 | 是否自动选择匹配到的答案 |
| 自动提交 | 选择答案后自动提交 |
| 倍速守护 | 视频加载后是否自动恢复设定倍速 |
| 自动恢复播放 | 视频暂停后是否自动恢复 |
| 远程题库 URL | 远程题库 JSON 文件的 URL |

## 从源码构建

```bash
npm install
npm run build
```

构建产物位于 `dist/` 目录。

## 技术栈

- Chrome Extension Manifest V3
- Rollup 模块打包
- 原生 JavaScript

## 项目结构

```
src/
  background.js         -- 后台 Service Worker
  content/              -- 内容脚本（注入到页面）
    index.js            -- 入口，初始化各模块
    autoAnswer.js       -- 自动答题逻辑
    videoControl.js     -- 视频控制逻辑
    taskExtractor.js    -- 任务列表提取
    logger.js           -- 日志发送
    state.js            -- 全局状态管理
  sidepanel/            -- 侧边栏 UI
  config/               -- 配置页面
  manual-select/        -- 手动选择答案弹窗
  lib/                  -- 公共库
    configManager.js    -- 配置管理
    questionBank.js     -- 题库管理
    utils.js            -- 工具函数
  shared/
    actions.js          -- 消息常量定义
```
